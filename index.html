About PDF Processor

This tool was originally designed to remove numerical marks from reports generated by the JumpRope assessment tracking software, helping teachers focus on qualitative feedback. However, it has evolved into a fully general-purpose PDF editor that allows you to redact content, add text overlays, or insert images into any batch of PDF files using a repeating template pattern.

PRO TIP: When generating reports in JumpRope, select the "Duplex" option. This ensures that every student report has the same number of pages (by inserting blank pages where necessary), preventing alignment issues during batch processing.

How to Use

  1. Create a Template: Go to the Template Editor tab and upload a sample PDF file (e.g., one student's report).
  2. Define Rules: Set the "Pages per Student". Draw boxes to redact areas, add text, or insert images.
  3. Set Patterns: Use the properties panel to choose how often these boxes repeat (e.g., "Per Student", "Every Page", "Rest of Report").
  4. Save Template: Templates autosave to your browser as you work — no save button needed. Give your template a name in the toolbar. Use ⬇ Backup in the toolbar or the 🗂️ Files panel to download a permanent .json file.
  5. Process Batch: Go to the PDF Processor tab. Load your saved template, verify the page count, and drag in all your student reports. The app will process each file automatically.

🗂️ Saving & Files

Templates can be saved in two ways. They work together but serve different purposes — it's important to understand both.

💾 Browser Storage Temporary

The editor autosaves your template to browser storage as you work — every time you draw, move, resize, or change a property, it saves automatically. You can rename your template at any time using the name field in the editor toolbar. The 🗂️ Files panel shows all your saved templates and lets you switch between them.

⚠️ This storage can disappear without warning — clearing browser data, switching browsers, using a different device, or even some browser updates will wipe it completely.

📁 JSON File Backup Permanent

⬇ Backup in the editor toolbar (or the ⬇ Backup button on each template in the Files panel) downloads a .json file to your computer. This is your real, durable backup — it works in any browser, on any device, and survives anything. Always download a JSON backup for any template you care about keeping.

⚠️ Don't rely on browser storage alone. "Saved to browser" is not the same as saved to a file. Browser storage is invisible and can be cleared at any time. If you've put real effort into a template, always download a .json backup — it takes one click and you'll never lose your work.
